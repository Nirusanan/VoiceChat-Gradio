# Voice Gradio-App

## Step 1: create and activate virtual environmnet

## Step 2: Install Requirements
`
pip install -r requirements.txt
`

## Step 3: Create .env File

`
GROQ_API_KEY=your_secret_key_here
`

## Step 4: Run the App
`
py app.py
`
